## Recommended Screenshot Specifications:
- Format: PNG or JPG
- Dimensions: 1200x800px (3:2 aspect ratio) or similar
- File size: Optimized, under 500KB each
- Content: Homepage or key feature screenshots

The images will automatically display in the portfolio cards when added to this directory.
